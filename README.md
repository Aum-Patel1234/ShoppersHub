# **ShoppersHub**

## Description
**ShoppersHub** is a web application that fetches and displays product information using a sample Products API. The project uses Express, MongoDB, and Mongoose for backend and database management. The primary goal is to deploy the website on the cloud for live access.

## Features
- Product listings with details such as name, price, description, and images.
- Categorized product views.
- Secure backend with data persistence.

## Technologies
- **Backend**: Node.js, Express
- **Database**: MongoDB, Mongoose
- **Security**: Helmet for security
- **Environment Management**: dotenv for environment variables
